Angebotskonfigurator – Sielaff Austria · Stand Juni 2026

Was ist dieses Projekt?

Ein webbasierter Angebotskonfigurator für Sielaff Austria GmbH. Außendienstmitarbeiter erfassen auf der Messe Kundendaten und wählen Produkte/Leistungen aus — das System generiert automatisch ein professionelles PDF-Angebot das per E-Mail verschickt werden kann.

Änderungen deployen
In GitHub Desktop: geänderte Dateien auswählen → kurze Commit-Nachricht → "Commit to main" → "Push origin". Render.com deployed automatisch innerhalb von 1–2 Minuten.
⚠️ Frontend-Dateien (.jsx) sind React/JavaScript — Grundkenntnisse nötig. Reine Text- und JSON-Änderungen (Changelog, Anleitung, Portal) sind ohne Programmierkenntnisse möglich.

🚨 Was tun wenn etwas nicht funktioniert?

ProblemWo nachschauen
App lädt nicht / zeigt Fehler Render Dashboard → Frontend Service → Logs
PDF wird nicht erstellt Render Dashboard → Backend Service → Logs
E-Mails kommen nicht an Resend.com → Logs · App Einstellungen → E-Mail Versand prüfen
Daten fehlen / Datenbank-Fehler Supabase Dashboard → Project → Logs oder Table Editor
Deployment fehlgeschlagen Render Dashboard → Service → Deploy Logs · GitHub → letzter Commit prüfen
App nach Änderung kaputt GitHub Desktop → "Revert" des letzten Commits → Push → Render rollt zurück
